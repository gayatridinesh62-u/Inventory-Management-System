# 📦 Inventory Management System
Inventory Management System using Python Flask , HTML , CSS , MYSQL

## 📖 Project Overview

The Inventory Management System is a web-based application developed using Python Flask and MySQL to efficiently manage inventory operations. The system is designed to help businesses or organizations maintain product records, manage stock levels, organize categories, and handle supplier information in a structured and user-friendly manner.

This project demonstrates full-stack development by integrating frontend technologies (HTML, CSS, JavaScript) with backend logic (Flask framework) and a relational database (MySQL).


## 🎯 Project Objectives

* To design a centralized system for managing inventory data.
* To implement complete CRUD operations for inventory management.
* To integrate a MySQL database with a Flask-based web application.
* To create a clean and customizable user interface using CSS.
* To demonstrate real-world database and backend development concepts.

## 🚀 Key Features

### 🔐 Authentication Module

* Secure admin login
* Session-based authentication
* Logout functionality

### 📊 Dashboard

* Displays total number of products
* Displays total number of categories
* Displays total number of suppliers
* Provides quick overview of inventory data

### 📂 Category Management

* Add new categories
* View all categories
* Delete categories

### 🚚 Supplier Management

* Add supplier details
* View supplier list
* Delete supplier records

### 📦 Product Management

* Add new products
* Edit product details
* Delete products
* Assign category and supplier to products
* Manage stock quantity and price
* Display joined data using relational database concept
